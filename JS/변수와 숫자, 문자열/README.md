# 변수

- 변수란 '데이터에 붙이는 이름표'이다. 변수를 이용하면 이름표를 붙여 둔 데이터를 기억해두었다가 필요할 때마다 재사용 할 수 있게 된다.

```javascript
let 변수이름 = 데이터;
```

## 변수의 선언과 초기화
- 만들어진 변수에 첫 데이터를 지정하는 작업 
```javascript
let 변수이름 ;      // 변수 선언
변수이름 = 데이터 ;     // 변수 초기화
```

# 변수 규칙

- 변수명에는 오직 문자와 숫자, 그리고 기호$과_만이 포함될 수 있다.
- 변수명의 첫 번째 글자로 숫자가 올 수 없다.
- 이미 다른 땃을 가지고 있는 단어(키워드)는 변수명으로 사용할 수 없다.
